## 🪴 项目运行

1. 拉取项目代码
```bash
git clone https://github.com/baiwumm/Xmw-Admin.git
cd Xmw-Admin
// 进入前端
cd Xmw_web
// 进入后端
cd Xmw_server
```

2. 安装依赖
```
npm install -g pnpm
pnpm install
```

- 开发模式运行
```
// 前端启动
pnpm start
// 后端启动
pnpm start:dev
```

- 编译项目
```
pnpm build
```
